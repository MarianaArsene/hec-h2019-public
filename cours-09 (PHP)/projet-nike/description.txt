Cours 9 - PHP - La base d'un langage serveur

Voici les étapes à réaliser :

1) Téléchargez les fichiers de départ dans le dossier /code-debut, ainsi que l'aperçu (apercu.jpg). 
    
Note : Nous mettrons de côté ce que nous avons développé jusqu'à maintenant, et nous commencerons à travailler sur une nouvelle section : "Vous aimerez peut-être aussi". 

2) Consultez l'aperçu et essayez par vous-mêmes de reproduire la page HTML.
    
======Vous êtes encouragés à réellement essayer de reproduire l'aperçu par vous-mêmes avant de lire les détails de construction. Cela vous permettra de pratiquer votre sens d'analyse.

3) À partir du fichier index.php, effectuez l'inclusion des fichiers requis (voir les notes à l'intérieur du fichier, qui donnent des indices...)
    
4) Dans le fichier souliers.php, ajoutez les deux lignes permettant de créer les souliers 3 et 4 (voir note dans fichier). Inspirez vous de l'aperçu pour le titre, la description, les coloris et le coût. 
    
5) Dans le fichier souliers, ajoutez les deux lignes permettant d'inclure les  nouveaux souliers à la variable $souliers (voir note dans fichier).
    
6) Dans le fichier vous-aimerez.html.php, ajoutez la commande permettant d'afficher le titre du soulier (voir la note dans le fichier). Inspirez-vous de la commande permettant d'afficher la quantité de coloris quelques lignes plus haut.
